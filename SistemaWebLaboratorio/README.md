# SistemaWebLaboratorio

Estructura del repositorio:

- documentos/: documentación del proyecto organizada por categorías.
- Software/: código fuente del sistema web (monorepo actual).

Convenciones:
- Nombres de carpetas sin acentos.
- Archivos `.env` no se versionan.

Cómo empezar:
- Entra a `Software/` y sigue las instrucciones del proyecto.
- Documenta el proyecto en `documentos/` usando las plantillas.
